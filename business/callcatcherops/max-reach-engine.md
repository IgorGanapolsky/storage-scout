# Max Reach Engine (Service Acquisition)

## Objective
Reach as many qualified local businesses as possible without manual campaign operations.

## Throughput Targets
- New leads/day: 120+
- Unique markets/week: 25+
- First-touch emails/day: 120
- Follow-up emails/day: 60
- Minimum qualified meetings/week: 15

## Automation Stack
1. Lead generation (Google Places + market rotation)
   - Script: `autonomy/tools/lead_gen_broward.py`
   - Market file: `autonomy/data/us_growth_markets.json`
2. Outreach execution
   - Entry: `autonomy/run.py`
   - Config: `autonomy/config.callcatcherops.json`
3. One-command cycle
   - Script: `autonomy/tools/run_growth_cycle.sh`

## Standard Daily Run
```bash
bash autonomy/tools/run_growth_cycle.sh 120
```

## Scaling Levers
- Increase daily lead cap: pass larger limit to `run_growth_cycle.sh`.
- Add categories using `LEADGEN_CATEGORIES` environment variable.
- Add markets by extending `autonomy/data/us_growth_markets.json`.
- Split market sets by region and run separate cursor keys.

## Safety Constraints
- Keep CAN-SPAM compliance and unsubscribe enforcement enabled.
- Do not send to blocked role inbox local-parts.
- Do not make ranking/revenue guarantees in outbound copy.

## Weekly Evidence
- Leads generated by state + category
- Delivered emails, bounce rate, opt-out rate
- Replies, booked calls, setup sprint purchases
- Cost per qualified meeting and closed client count
