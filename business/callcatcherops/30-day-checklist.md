# 30-Day Checklist

**Week 1 — Asset Build + Initial Outreach**
1. Finalize landing page copy and publish.
2. Configure email deliverability (SPF, DKIM, DMARC).
3. Prepare lead list of 200 local businesses (CSV).
4. Run first outreach batch (30/day).
5. Send first baseline results to any replies.

**Week 2 — Pipeline + Proof**
1. Run daily outreach batches (30/day).
2. Log replies and move qualified leads to “baseline scheduled.”
3. Create 1 demo case study (even hypothetical, with clear assumptions).
4. Add 1 testimonial placeholder section on landing page.
5. Start weekly performance report template.

**Week 3 — Close + Delivery**
1. Book and run 3 baselines.
2. Convert 1 baseline into a QuickStart Build.
3. Document build steps and templates.
4. Record 1 short Loom demo (3–5 min).
5. Add Loom link to follow-up emails.

**Week 4 — Retainer + Scale**
1. Convert 1 QuickStart client into Managed Growth.
2. Increase outreach to 50/day if reply rates stay above 5%.
3. Add additional niche-specific landing page section.
4. Publish 2 social posts summarizing results.
5. Create a referral offer for existing clients.
